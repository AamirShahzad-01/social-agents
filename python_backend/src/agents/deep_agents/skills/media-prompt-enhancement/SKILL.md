# Media Prompt Enhancement - Enterprise Brand Quality

Comprehensive guide for enhancing AI generation prompts to enterprise brand standards.

## Purpose

Transform basic user prompts into professional, commercial-quality prompts for:
- **Image Generation**: Google Imagen, OpenAI GPT Image, Gemini Pro
- **Video Generation**: Google Veo, OpenAI Sora, Runway Gen-4

## Core Principle

**Preserve the user's main idea → Enhance with professional details → Return only the enhanced prompt**

---

## Enhancement Rules

### PRESERVE (Never Change)
- User's main subject (product, scene, concept)
- Specific brand names or product names mentioned
- Core visual concept they described
- Intended mood or feeling

### ENHANCE (Apply These)
1. **Subject Clarity** - Add specific details to main subject
2. **Setting/Context** - Define environment, background, atmosphere
3. **Lighting** - Professional lighting description
4. **Composition** - Camera angle, framing, depth of field
5. **Style** - Photography/cinematography keywords
6. **Quality Modifiers** - Professional, commercial terms
7. **Aspect Ratio** - Appropriate for intended use

---

## Brand Quality Standards

Apply enterprise brand aesthetics (Nike, Zara, L'Oréal quality):

| Standard | Description |
|----------|-------------|
| Composition | Clean, uncluttered, product-focused |
| Lighting | Professional studio or cinematic |
| Quality | Commercial/advertising grade |
| Colors | Sophisticated, brand-appropriate palettes |
| Mood | Premium, aspirational |

### Avoid
- Amateur or casual aesthetics
- Cluttered backgrounds
- Harsh or unflattering lighting
- Generic stock photo look

## Image Prompt Structure

```
[Subject with specific details], [setting/environment], [lighting description], [camera/composition], [style keywords], [quality modifiers], [aspect ratio]
```

### Aspect Ratios for Images

| Ratio | Use Case |
|-------|----------|
| 1:1 | Instagram feed, product squares |
| 4:5 | Instagram optimal, feed posts |
| 9:16 | Stories, Reels covers, TikTok |
| 16:9 | Website banners, YouTube thumbnails |
| 3:2 | Campaign photography |

### Lighting Keywords
- soft studio lighting
- dramatic side lighting
- natural window light
- golden hour warmth
- rim light accent
- beauty lighting (for cosmetics)
- professional product lighting

### Quality Modifiers
- professional photography
- commercial advertising quality
- high resolution
- premium brand aesthetic
- 4K detail
- sharp focus

## Video Prompt Structure (Shot-Based)

**Important:** AI video generators create 5-10 second clips. For longer videos, break into multiple 7-second shots.

### Single Shot Format (5-10 sec)
```
[Camera movement]: [Subject with action]. [Setting]. Lighting: [description]. Style: [cinematography style]. [Aspect ratio].
```

### Multi-Shot Format (15-60 sec videos)
```
**Shot 1** (0-7 sec): [Camera]: [Action]. [Setting]. [Lighting]. [Style]. [Ratio].
**Shot 2** (7-14 sec): [Camera]: [Action]. [Setting]. [Lighting]. [Style]. [Ratio].
**Shot 3** (14-21 sec): [Camera]: [Action]. [Setting]. [Lighting]. [Style]. [Ratio].
(Continue for total duration)
```

### Duration to Shots Reference

| Video Duration | Number of Shots |
|----------------|-----------------|
| 7-10 sec | 1 shot |
| 15 sec | 2 shots |
| 21 sec | 3 shots |
| 30 sec | 4-5 shots |
| 60 sec | 8-9 shots |

### Aspect Ratios for Video

| Ratio | Use Case |
|-------|----------|
| 9:16 | Reels, TikTok, Shorts, Stories |
| 16:9 | YouTube, commercials, website |
| 1:1 | Social feed video, product loops |
| 4:5 | Instagram feed video |

### Camera Movement Keywords
- slow dolly in/out
- smooth tracking shot
- elegant orbit
- subtle crane up/down
- controlled pan reveal
- steady push in

### Style Keywords
- commercial production quality
- cinematic brand film
- premium advertising footage
- professional cinematography
- editorial fashion film

## Provider-Specific Optimization

### Google Imagen 4
- Lead with subject, then context, then style
- Include specific photography modifiers
- Add quality terms at end
- Specify aspect ratio explicitly

### Google Veo
- Use director's note format
- Include camera movement first
- Describe lighting and audio
- Specify duration if needed

### OpenAI Sora
- Lead with style keyword
- Describe action clearly
- Include camera angle
- Add cinematic quality terms

### Runway Gen-4
- Use positive phrasing only
- Lead with camera movement
- Vivid motion verbs
- Keep concise but detailed

### OpenAI GPT Image
- Natural language works well
- Be conversational but specific
- Include mood and atmosphere
- Add quality modifiers

## Format-Specific Guidance

### For Reels/Shorts (9:16) - 15-30 seconds
- 2-4 shots of 7 seconds each
- Vertical framing, subject centered
- Quick, attention-grabbing visuals
- Product as hero

### For Feed Posts (4:5, 1:1)
- Clean, premium compositions
- Product hero shots
- Lifestyle context
- Editorial quality

### For Carousel Content
- Consistent visual style across images
- Multiple angles or features
- Educational or feature-focused

### For Stories
- Bold, immediate visual impact
- Vertical, mobile-first
- Text-overlay friendly compositions

✅ Return ONLY the enhanced prompt text

✅ For videos: Break into 7-second shots
✅ No explanations or commentary
✅ No "Here's your prompt:" prefix
